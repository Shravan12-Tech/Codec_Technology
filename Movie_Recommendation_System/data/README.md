# 🎬 Movie Recommendation System

A Python-based Movie Recommendation System with a modern GUI built using CustomTkinter. The application recommends similar movies based on movie metadata and machine learning techniques.

## 📌 Features

- Modern Dark Theme GUI
- Movie Recommendation Engine
- Interactive User Interface
- Dataset-Based Recommendations
- Easy-to-Use Search Functionality
- Python Desktop Application

## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-Learn
- CustomTkinter
- Machine Learning
- CSV Dataset


```bash
python gui.py
```

## 📊 Dataset

This project uses the TMDB 5000 Movie Dataset.

Files:

- tmdb_5000_movies.csv
- tmdb_5000_credits.csv

## 💻 How It Works

1. User enters a movie name.
2. The system searches the dataset.
3. Similar movies are identified using machine learning algorithms.
4. Recommended movies are displayed in the GUI.
